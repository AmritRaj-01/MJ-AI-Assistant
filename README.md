# 🤖 MJ AI Assistant

MJ AI Assistant is a Python-based desktop voice assistant powered by Google's Gemini AI.  
It can listen to voice commands, open websites, launch applications, search Google, tell time/date, and respond intelligently using AI.

---

## 🚀 Features

- 🎤 Voice Recognition
- 🤖 Gemini AI Chat Support
- 🌐 Open Websites using Voice Commands
- 🔍 Google Search
- 🖥️ Open Desktop Applications
- 🕒 Time & Date Information
- 🔊 Text-to-Speech Responses

---

## 🛠️ Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Google Gemini API

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone
```

### 2️⃣ Move into Project Folder

```bash
cd MJ-AI-Assistant
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Add Your Gemini API Key

Open `mj.py` and replace:

```python
API_KEY = "YOUR_API_KEY"
```

with your actual API key.

---

## ▶️ Run the Project

```bash
python mj.py
```
